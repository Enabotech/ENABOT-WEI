# ENABOT-YAO
第一代个人机械臂。

enabotech@126.com

## 电气
### 电机及驱动
* 东方减速步进电机套件。
### 运动控制器
* Galil DMC-2183/DMC-4080。
* 目前正在开发基于ODRIVE和VESC的开源无刷电机驱动及控制单元。
### 主控单元
* 基于Ubuntu + ROS Kinetic的PC系统。
## 机械
* 已完成设计及结构验证。主要参考ABB IRB 14000机械臂自行设计。
## 软件
* 完成基于ROS的MOVEIT+GAZEBO仿真系统搭建。
* 初步完成系统框架程序。
